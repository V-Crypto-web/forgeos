# 🚀 Hiring Brief: ForgeOS Founding Team
*Location: Global/Remote or Hub (San Francisco/European Hub)*

## About ForgeOS
ForgeOS is not an AI-IDE. We are building an **Autonomous Development Infrastructure Platform** (PaaS). We treat software development as computation. Our platform features an Intelligence Supply Layer, an Anti-loop Heuristic Governor, and Enterprise-grade Multi-Tenant Governance. We are aiming for a $10B+ market cap by fundamentally replacing routine engineering tasks with deterministic, auditable, and self-improving AI worker clusters.

## The Team Profile
We are looking for **3-4 highly senior Founding Engineers** who are deeply passionate about autonomous agents, deterministic sandboxing, and enterprise architecture.

---

### 1. Founding AI / ML Systems Architect (The Orchestrator)
**The Role:** You will own the *Intelligence Supply Layer* and the *Core State Machine*. You are responsible for making sure LangGraph orchestrates Planner, Coder, and Verifier brains efficiently, without infinite loops, leveraging OpenAI, Anthropic, and open-weight models interchangeably.
**Key Requirements:**
- Deep expertise in AI Agent Frameworks (LangGraph, Pydantic AI, AutoGen).
- Systems architecture background (You aren't just calling APIs; you build state machines).
- Experience with LLM routing, token economy optimization, and context-window compression.

### 2. Founding Infrastructure & Security Engineer (The Sandbox)
**The Role:** You will own the *Execution Sandboxes*, *Policy Enforcement Layer*, and *Secrets Vault*. ForgeOS needs to execute untrusted code generated by AI safely. You will build the robust multi-tenant cloud infrastructure and the strict deterministic toolsets.
**Key Requirements:**
- Deep expertise in Docker SDK, Containerization, and macOS Process Restrictions.
- Experience with ASTs (Tree-sitter), static analysis, and secure code execution.
- Background in Cloud Infrastructure (Multi-tenant DBs, K8s, API Gateways, Secret Vaults).

### 3. Founding Backend & Integrations Engineer (The Connectors)
**The Role:** You will own the *Developer Platform API* and the *Connector Layer*. You will ensure ForgeOS natively integrates into GitHub, Jira, CI/CD, and Observability stacks (OpenTelemetry, Datadog), making it enterprise-ready on day one.
**Key Requirements:**
- Expert in FastAPI, gRPC, and building scalable Public APIs.
- Extensive experience with GitHub Apps/APIs, Webhooks, CI/CD pipelines.
- Deep understanding of Telemetry (OTel, Grafana, Tracing).

### 4. Founding Full-Stack / WebGL Engineer (The Interface)
**The Role:** You will own *Forge Desktop* and the *Observability Game Mode (DevSim)*. You will build the Tauri-based native shell for the CTO to directly manage the active agent clusters, eventually visualizing this in a Three.js WebGL SimCity-style interface.
**Key Requirements:**
- Mastery of modern React, TypeScript, and state management.
- Experience building Native Desktop apps via Tauri or Electron.
- Experience with Three.js / WebGL / React Fiber (for data streaming visualization).
- Sharp product sense for developer UX (you understand what a CTO wants to see).
